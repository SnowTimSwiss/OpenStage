# OpenStage 🎬

Open source presentation software for live events, worship services, and concerts.

Built for the people running the tech on stage – not just the speakers.

## Features (planned)
- 🖥️ Multi-display configuration (operator view + beamer output)
- 📝 Song lyrics – slide by slide
- ⏱️ Countdown timer
- 🎬 Video playback
- 🖼️ Image & slide queue (drag & drop)
- 🎵 Music playback with queue
- ⬛ One-click blackout

## Tech Stack
- [Tauri](https://tauri.app) (Rust + TypeScript)
